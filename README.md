</div>

Sensei is a simple command line tool to open documentation for any crate in crates.io. 

## Usage 

```
sensei <crate> -v <version>
```

#### Flags
```
-h, --help       Prints help information
-V, --version    Prints version information
``` 
## How to install?
```
cargo install sensei
```
