SENSEI (先生)

Sensei is a simple command line tool to open documentation for any crate in crates.io.

Installation

cargo install sensei

Usage

sensei <crate> [OPTIONS]

Options

-v, --version <version>

Opens documentation for a specific version.

-q, --query <query>

Specifies query to search documentation.

Flags

-h, --help

Prints help information.

Example

sensei serde -v 0.8.8 -q Serialize
